# ספר המתכונים של אביבה 📖

## הגדרה

### 1. Supabase
הרץ את `supabase-setup.sql` ב-SQL Editor של Supabase.
צור Storage bucket בשם `recipe-images` עם הרשאות ציבוריות.

### 2. משתני סביבה ב-Vercel
```
NEXT_PUBLIC_SUPABASE_URL=...
NEXT_PUBLIC_SUPABASE_ANON_KEY=...
ANTHROPIC_API_KEY=...
NEXT_PUBLIC_ADMIN_PASSWORD=aviva2024
```

### 3. הרצה מקומית
```bash
npm install
npm run dev
```
